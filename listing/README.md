# Play Store Listing Content

This directory contains all the content needed for publishing the Ollama AI Chat app on the Google Play Store.

## Files Overview

- **app_name.txt** - The app name (30 characters max)
- **short_description.txt** - Short description (80 characters max)
- **full_description.txt** - Full app description (4000 characters max)
- **promotional_text.txt** - Promotional text (80 characters max, optional)
- **whats_new.txt** - What's New section for the first release
- **keywords.txt** - Suggested keywords for ASO (App Store Optimization)
- **category.txt** - Recommended app categories
- **screenshots_guide.txt** - Detailed guide for creating screenshots
- **feature_graphic.txt** - Guidelines for creating the feature graphic

## Character Limits

- **App Name**: 30 characters
- **Short Description**: 80 characters
- **Full Description**: 4000 characters
- **Promotional Text**: 80 characters (optional)
- **What's New**: 500 characters

## Usage Instructions

1. Copy the content from each file into the corresponding Play Console field
2. Follow the screenshots guide to create required screenshots
3. Create a feature graphic following the feature_graphic.txt guidelines
4. Review all content for accuracy before publishing
5. Consider A/B testing different descriptions to optimize downloads

## Notes

- All content is optimized for the Google Play Store
- Keywords are suggestions - use them naturally in descriptions
- Screenshots should be high-quality and showcase real app functionality
- Feature graphic must be exactly 1024 x 500 pixels
- Ensure all text complies with Google Play policies
